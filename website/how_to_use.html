Get font dimensions for your font

To get font dimensions go to Squeeze.js font generator and upload your font, set your parameters and click on "Load font table". Set your file name and your variable name then click on "Get dimensions".

Include font dimensions file and Squeeze.js in your page

Include the .js file you got and Squeeze.js file in your project.

Run script

var font_dimensions = "variable that you defined in font dimensions generator";
var box_height 		= 100;
var box_width 		= 400;
var words 			= "Lorem ipsum dolor";

var squeeze = new Squeeze(font_dimensions, box_height, box_width, words);
var response = squeeze.getBestFontSize();

var line_height = response.line_height;
var font_size = response.font_size
Squeeze.js only calculates the best font size and font height. You still have to manually change the CSS afterwards.
